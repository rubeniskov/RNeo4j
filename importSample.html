importSampleR Documentation

Import Sample Datasets

Description

Populate the graph database with one of the sample datasets supplied with this package.

Usage

importSample(graph, data, input = TRUE)

Arguments

graph

A graph object.

data

The dataset to be imported. Accepts a string.

input

Whether or not confirmation (in the form of external input) from the user should be required.

Details

Datasets available include tweets, dfw, caltrain and movies. For more information on these datasets, see here: http://nicolewhite.github.io/RNeo4j/samples/

Examples

graph = startGraph("http://localhost:7474/db/data/")

importSample(graph, "tweets")

summary(graph)
getConstraint(graph)
